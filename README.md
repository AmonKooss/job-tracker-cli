# Job Tracker CLI

Gerenciador de vagas em terminal feito com Python e SQLite para praticar CRUD e integração com banco de dados.

Funcionalidades:
- Adicionar vaga
- Listar vagas
- Remover vaga
- Buscar vagas por status
- Atualizar status da vaga

Tecnologias utilizadas:
- Python
- SQLite


## How to run

```bash
python main.py
```
