# 🗂️ Job Tracker CLI

Ferramenta de linha de comando para gerenciar candidaturas de emprego,
desenvolvida em Python com SQLite.

Criada como projeto prático para consolidar conhecimentos em
CRUD, modelagem de banco de dados e organização de código em camadas.

---

##  Funcionalidades

-  Adicionar nova vaga (empresa, cargo, link, status)
-  Listar todas as vagas cadastradas
-  Filtrar vagas por status (ex: "enviado", "entrevista", "recusado")
-  Atualizar o status de uma vaga
-  Remover vaga

---

##  Tecnologias

| Tecnologia | Uso |
|---|---|
| Python 3.x | Linguagem principal |
| SQLite | Banco de dados local |
| SQLAlchemy | ORM para modelagem e queries |

---

##  Como executar

**Pré-requisitos:** Python 3.8+

```bash
# Clone o repositório
git clone https://github.com/AmonKooss/job-tracker-cli.git
cd job-tracker-cli

# Execute
python main.py
```
